Block Element

  1. What are block Element?

  • Block Elements are those which always starts from new line.
  • It always takes full width available.
  • For example:
    1. P
    2. h1
    3. h6
    4. div

Inline Element

  1. What are Inline Element?

  • Inline Elements are those which always starts from next point where previous Elements ended.
  • It always takes space required by content.
  • For example:
    1. img
    2. span
    3. button
    4. input
